# OpenAI Build Week submission evidence

This directory preserves the public, non-secret evidence used to produce the
JAS — JAH Action Script PHP demo for OpenAI Build Week.

## Published deliverables

- Devpost: <https://devpost.com/software/jas-jah-action-script-php-7t48ci>
- Public video: <https://youtu.be/r9wakYm8Y9Q>
- Category: Developer Tools
- Video duration: 2 minutes 53 seconds
- Devpost submission ID: `1098042`

The private `/feedback` Session ID is intentionally stored only in the required
Devpost field and is not part of this repository.

## Video evidence

- `video/EDIT_PLAN.md` records the editing, accuracy and safety decisions.
- `video/VOICEOVER_EN_BUILD_STORY.txt` is the final English narration.
- `video/subtitles_build_story_en.srt` contains the final English subtitles.

Temporary concatenation manifests, intermediate narration and intermediate
subtitle revisions are local production artifacts and are intentionally not
versioned.

The final video shows a real 83-minute-and-49-second recording containing an
81-minute Codex build of Taquería Viva. It explains that Codex with GPT-5.6 SOL
and Medium reasoning effort accelerated development and review but is not a JAS
runtime dependency.
