# JAS 2.0.0

JAS 2.0 congela la primera API pública estable para aplicaciones PHP organizadas,
tipadas y gobernadas. Incluye DataCore autoritativo, identidad institucional,
runtime de acciones, Web seguro, colas, continuidad, diagnósticos y tooling.

La entrega incorpora `examples/reference_portal`, una aplicación funcional con
ocho dominios, roles/capacidades, datos cifrados, feed y moderación separados,
mensajería, notificaciones, auditoría y restauración verificada.

Los módulos distribuidos, observabilidad externa, SDK/bridge LSP y adaptadores de
identidad federada permanecen experimentales. La revisión criptográfica y el
penetration test independientes continúan pendientes y no se presentan como
certificación del proyecto.

Consulte `docs/API_STATUS.md` y `docs/JAS_2_0_MIGRATION.md` antes de actualizar.
